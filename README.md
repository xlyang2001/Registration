# 融合局部与全局特征的单椎体2D/3D配准网络
## Prerequisites
- `Pytorch `
- `Cuda`
- `Numpy`

This code was tested with `Pytorch` and `NVIDIA GeForce RTX 3090 GPU`.

`test_reg.py`

Five preprocessed DRR images can be downloaded with  `Data/Drr_9000.png`
